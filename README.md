# CyberCases: Cipher High

A small story-driven cybersecurity game prototype built with Python and Pygame.

## Current Version

- Terminal-like interface with a cyber-theme
- Three short playable incident-response cases
- Keyboard-friendly terminal input with cursor movement and command history
- Case summaries, game scoring, and a final results screen

## Run It

```bash
python main.py
```

## Commands

- `help`
- `files`
- `open <file>`
- `decode <text>`
- `rot13 <text>`
- `submit <answer>`
- `status`
- `clear`

## Input Controls

- Hold `Backspace` to delete continuously
- Use `Left` and `Right` to move through text
- Use `Home` and `End` to jump in the prompt
- Use `Up` and `Down` to browse command history


## Project Direction

This version is intentionally scoped to a polished mini-game:

1. Learn the story from the case file.
2. Inspect evidence for each incident.
3. Use the terminal to decode clues and submit answers.
4. Clear all three cases and review the campaign summary.

The next steps to add are sound effects, small animations, and more visual flavor in the transitions, and longer, more elaborate cases.
